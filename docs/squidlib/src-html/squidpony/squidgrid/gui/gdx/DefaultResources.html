013/**
014 * Default BitmapFonts, a sample image, "stretchable" fonts and icons, and a central RNG for use with LibGDX.
015 * All but one of the resources here need to be downloaded separately, to avoid bloating the size of the dependency jar
016 * with many fonts that a game probably won't use. You can download assets in a jar with a release of SquidLib, or
017 * individually from https://github.com/SquidPony/SquidLib/tree/master/assets . The assets you choose should go in the
018 * android/assets (core/assets if you don't have an Android project) subfolder of your game if you use the normal libGDX
019 * setup tool or SquidSetup, or the assets subfolder if you use czyzby's alternate setup tool,
020 * https://github.com/czyzby/gdx-setup . They should not be placed in subdirectories of assets, otherwise this won't be
021 * able to find them.
022 * <br>
023 * The fonts provided are mostly monospaced, with most looking rather similar (straight orthogonal lines and right-angle
024 * curves), but some that look... better than the rest. For monospaced fonts that could be used on a grid in SquidPanel
025 * or SquidLayers, one good choice is Inconsolata-LGC, accessible by getSmoothFont(), getLargeSmoothFont(), (ideally for
026 * narrow glyphs) getStretchableFont(), or (ideally for square glyphs) getStretchableSquareFont(). Another is Computer
027 * Modern, accessible by getStretchableTypewriterFont(), which has a chunky, ornamented style. For variable-width fonts
028 * that can be used in TextPanel and LinesPanel, among others, you can use the high-quality serif font Gentium,
029 * accessible by getStretchablePrintFont(), or Noto Sans, a clean sans font accessible by getStretchableCleanFont(). All
030 * of the fonts so far support the Latin, Greek, and Cyrillic alphabets.
031 * <br>
032 * If you can't decide, go with getStretchableFont() or getStretchableSquareFont(), which return TextCellFactory
033 * objects, and call their .width(int), .height(int), and .initBySize() methods to make them the size (and aspect ratio)
034 * you want. You can use getStretchableFont() or getStretchableTypewriterFont() with the same width and height to make
035 * a horizontally-stretched version of a square font instead of the existing square fonts that add blank space.
036 * <br>
037 * The most Latin script support for a monospaced font is in the font Mandrill, accessible by getDefaultUnicodeFont()
038 * and getLargeUnicodeFont() in two different sizes, and the latter should be suitable for everything from Spanish and
039 * French, to Polish to Vietnamese. You can use Gentium for even better language support with a variable-width font in
040 * LinesPanel or the like; it is accessible with getStretchablePrintFont(). Google's Noto font also supports many glyphs
041 * in a variable-width format; it is accessible with getStretchableCleanFont() but may have some issues with baseline
042 * level changing and character-to-character alignment seeming too high or low.
043 * <br>
044 * The sample image is a tentacle taken from a public domain icon collection graciously released by Henrique Lazarini;
045 * it's fitting for SquidLib to have a tentacle as a logo or something, I guess?
046 * <br>
047 * The icons are from http://game-icons.net , there are over 2000 of them, and they're among the only images that should
048 * both be recolor-able easily (like chars in a font) and resize-able easily if you use a stretchable font at the same
049 * time. The stretchable font TextCellFactories do some work with shaders that makes most images look incorrect, but
050 * allow distance field fonts and these distance field icons to resize smoothly. Mixing stretchable fonts with
051 * non-stretchable images or fonts is not a great idea, and usually involves a slow shader change back and forth.
052 * <br>
053 * You can get a default RNG with getGuiRandom(); this should probably not be reused for non-GUI-related randomness,
054 * but is meant instead to be used wherever randomized purely-aesthetic effects are needed, such as a jiggling effect.
055 * Created by Tommy Ettinger on 7/11/2015.
056 */

215    static BitmapFont copyFont(BitmapFont font)
216    {
217        return new BitmapFont(new BitmapFont.BitmapFontData(font.getData().getFontFile(), false),
218                font.getRegions(), font.usesIntegerPositions());
219    }
220    private DefaultResources()
221    {
222        if(Gdx.app == null)
223            throw new IllegalStateException("Gdx.app cannot be null; initialize GUI-using objects in create() or later.");
224        Gdx.app.addLifecycleListener(this);
225    }
