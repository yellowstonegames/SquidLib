010/**
011 * Some classes need detailed information about what cells are considered adjacent to other cells, and may
012 * need to construct a customized mapping of cells to their neighbors. Implementations of this abstract
013 * class provide information about all sorts of things, including the distance metric (from DijkstraMap),
014 * but also the maximum number of states that can be moved to in one step (including rotations at the same
015 * point in space, in some cases), and whether the type of map uses a "two-step" rule that needs two
016 * sequential moves in the same direction to be viable and unobstructed to allow movement (which is
017 * important in thin-wall maps).
018 * <br>
019 * When CustomDijkstraMap and similar classes need to store more information about a point than just its
020 * (x,y) position, they also use implementations of this class to cram more information in a single int.
021 * This abstract class provides methods to obtain four different numbers from a single int, though not all
022 * implementations may provide all four as viable options. It also provides a utility to get a Coord from an
023 * int. X and Y are exactly what they always mean in 2D Coords, R is typically used for rotation, and N is
024 * typically used for anything else when it is present. The convention is to use N for the Z-axis when
025 * elevation/depth should be tracked, or for any more specialized extensions to the information carried at
026 * a point. The composite() method produces a compressed int from X, Y, R, and N values, and the validate()
027 * method allows code to quickly check if an int is valid data this class can use. Other information is
028 * tracked by fields, such as height, width, rotations, and depths, where the maximum number of possible
029 * states is given by height * width * rotations * depths, and the minimum for any of these int fields is 1.
030 * <br>
031 * Lastly, the neighborMaps() method produces very important information about what neighbors each cell has,
032 * and by modifying the returned int[][], you can produce "portal" effects, wraparound, and other useful
033 * concepts. The value it returns consists of an array (with length == maxAdjacent) of arrays (each with the
034 * same size, length == width * height * rotations * depth). The values in the inner arrays can be any int
035 * between 0 and (width * height * rotations * depth), which refers to the index in any of the inner arrays of
036 * a neighboring cell, or can be -1 if there is no neighbor possible here (typically at edges or corners of the
037 * map, some of the neighbors are not valid and so use -1). In normal usage, a for loop is used from 0 to
038 * maxAdjacent, and in each iteration the same index is looked up (the current cell, encoded as by composite()
039 * or obtained as an already-composited neighbor earlier), and this normally gets a different neighbor every
040 * time. In methods that do a full-map search or act in a way that can possibly loop back over an existing cell
041 * in the presence of wrapping (toroidal or "modulus" maps) or portals, you may want to consider tracking a
042 * count of how many cells have been processed and terminate any processing of further cells if the count
043 * significantly exceeds the number of cells on the map (terminating when 4 times the cell count is reached may
044 * be the most extreme case for very-portal-heavy maps).
045 * Created by Tommy Ettinger on 8/12/2016.
046 */
047public abstract class Adjacency implements Serializable {
048    private static final long serialVersionUID = 0L;
049    /**
050     * The array of all possible directions this allows, regardless of cost.
051     */
052    public Direction[] directions;
053    /**
054     * The maximum number of states that can be considered adjacent; when rotations are present and have a
055     * cost this is almost always 3 (move forward, turn left, turn right), and in most other cases this is
056     * 4 (when using Manhattan distance) or 8 (for other distance metrics).
057     */
058    public int maxAdjacent;
059    /**
060     * Only needed for thin-wall maps; this requires two steps in the same direction to both be valid moves
061     * for that direction to be considered, and always moves the pathfinder two steps, typically to cells
062     * with even numbers for both x and y (where odd-number-position cells are used for edges or corners
063     * between cells, and can still be obstacles or possible to pass through, but not stay on).
064     */
065    public boolean twoStepRule;
066    /**
067     * If you want obstacles present in orthogonal cells to prevent pathfinding along the diagonal between them, this
068     * can be used to make single-cell diagonal walls non-viable to move through, or even to prevent diagonal movement if any
069     * one obstacle is orthogonally adjacent to both the start and target cell of a diagonal move.
070     * <br>
071     * If this is 0, as a special case no orthogonal obstacles will block diagonal moves.
072     * <br>
073     * If this is 1, having one orthogonal obstacle adjacent to both the current cell and the cell the pathfinder is
074     * trying to diagonally enter will block diagonal moves. This generally blocks movement around corners, the "hard
075     * corner" rule used in some games.
076     * <br>
077     * If this is 2, having two orthogonal obstacles adjacent to both the current cell and the cell the pathfinder is
078     * trying to diagonally enter will block diagonal moves. As an example, if there is a wall to the north and a wall
079     * to the east, then the pathfinder won't be able to move northeast even if there is a floor there.
080     * <br>
081     * A similar effect can be achieved with a little more control by using thin walls, where the presence of
082     * a "thin corner" can block diagonal movement through that corner, or the absence of a blocking wall in
083     * a corner space allows movement through it.
084     */
085    public int blockingRule;
086    /**
087     * This affects how distance is measured on diagonal directions vs. orthogonal directions. MANHATTAN should form a
088     * diamond shape on a featureless map, while CHEBYSHEV and EUCLIDEAN will form a square. EUCLIDEAN does not affect
089     * the length of paths, though it will change the DijkstraMap's gradientMap to have many non-integer values, and
090     * that in turn will make paths this finds much more realistic and smooth (favoring orthogonal directions unless a
091     * diagonal one is a better option).
092     */
093    public Measurement measurement;
094    /**
095     * Can be changed if the map changes; you should get the neighbors from neighborMaps() again after changing this.
096     */
097    public int width,
098    /**
099     * Can be changed if the map changes; you should get the neighbors from neighborMaps() again after changing this.
100     */
101    height,
102    /**
103     * Can be changed if the map changes; you should get the neighbors from neighborMaps() again after changing this.
104     */
105    rotations,
106    /**
107     * Can be changed if the map changes; you should get the neighbors from neighborMaps() again after changing this.
108     */
109    depths;
110
111    protected boolean standardCost = true;
112
113    public boolean hasStandardCost()
114    {
115        return standardCost;
116    }
117    /**
118     * Used in place of a double[][] of costs in CustomDijkstraMap; allows you to set the costs to enter tiles (via
119     * {@link #addCostRule(char, double)} or {@link #addCostRule(char, double, boolean)} if the map has rotations).
120     * A cost of 1.0 is normal for most implementations; higher costs make a movement harder to perform and take more
121     * time if the game uses that mechanic, while lower costs (which should always be greater than 0.0) make a move
122     * easier to perform. Most games can do perfectly well with just 1.0 and 2.0, if they use this at all, plus possibly
123     * a very high value for impossible moves (say, 9999.0 for something like a submarine trying to enter suburbia).
124     * <br>
125     * You should not alter costRules in most cases except through the Adjacency's addCostRule method; most Adjacency
126     * implementations will set a flag if any cost is set through addCostRule that is different from the default, and
127     * this flag determines early-stop behavior in pathfinding (it can be checked with {@link #hasStandardCost()}, but
128     * cannot be set directly).
129     * <br>
130     * Adjacency implementations are expected to set a reasonable default value for when missing keys are queried, using
131     * {@link IntDoubleOrderedMap#defaultReturnValue(double)}; there may be a reason for user code to call this as well.
132     */
133    public IntDoubleOrderedMap costRules = new IntDoubleOrderedMap(32);

144     * Encodes up to four components used by this Adjacency, putting them into one int.
145     * Returns -1 if the encoded position is out of bounds or otherwise invalid, otherwise any int is possible.
146     * You can get the individual values with {@link #extractX(int)}, {@link #extractY(int)}, {@link #extractR(int)},

151     * @param n the bonus component to encode; this can be used for height or other extra data in some implementations
152     * @return the encoded position as an int; -1 if invalid, non-negative for valid positions
153     */
154    public abstract int composite(int x, int y, int r, int n);

200    public String show(int data)
201    {
202        if(data < 0)
203            return "(-)";
204        if(rotations <= 1)
205        {
206            if(depths <= 1)
207                return "(" + extractX(data) + ',' + extractY(data) + ')';
208            return "(" + extractX(data) + ',' + extractY(data) + ',' + extractN(data) + ')';
209        }
210        if(depths <= 1)
211            return "(" + extractX(data) + ',' + extractY(data) + ',' + extractR(data) + ')';
212        return "(" + extractX(data) + ',' + extractY(data) + ',' + extractR(data) + ',' + extractN(data) + ')';
213    }
214    public String showMap(int[] map, int r)
215    {
216        r %= rotations;
217        StringBuilder sb = new StringBuilder(width * height * 8);
218        for (int y = 0; y < height; y++) {
219            for (int x = 0; x < width; x++) {
220                sb.append(show(map[(y * width + x) * rotations + r])).append(' ');
221            }
222            sb.append('\n');
223        }
224        return sb.toString();
225    }

227    public static class BasicAdjacency extends Adjacency implements Serializable {
228        private static final long serialVersionUID = 0L;
229
230        private BasicAdjacency() {
231            this(20, 20, Measurement.MANHATTAN);
232        }
233
234        public BasicAdjacency(int width, int height, Measurement metric) {
235            this.width = width;
236            this.height = height;
237            rotations = 1;
238            depths = 1;
239            measurement = metric;
240            if(metric == Measurement.MANHATTAN)
241            {
242                directions = Direction.CARDINALS;
243                maxAdjacent = 4;
244                invertAdjacent = new int[]{1, 0, 3, 2};
245            }
246            else
247            {
248                directions = Direction.OUTWARDS;
249                maxAdjacent = 8;
250                invertAdjacent = new int[]{1, 0, 3, 2, 7, 6, 5, 4};
251            }
252            twoStepRule = false;
253            blockingRule = 2;
254            costRules.defaultReturnValue(1.0);
255        }
256
257        @Override
258        public int extractX(int data) {
259            return data % width;
260        }
261
262        @Override
263        public int extractY(int data) {
264            return data / width;
265        }
266
267        @Override
268        public int extractR(int data) {
269            return 0;
270        }
271
272        @Override
273        public int extractN(int data) {
274            return 0;
275        }
276
277        @Override
278        public int composite(int x, int y, int r, int n) {
279            if(x < 0 || y < 0 || x >= width || y >= height)
280                return -1;
281            return y * width + x;
282        }
283
284        @Override
285        public boolean validate(int data) {
286            return data >= 0 && extractY(data) < height;
287        }
288
289        @Override
290        public int[][][] neighborMaps() {
291            int[][][] maps = new int[2][maxAdjacent][width * height * rotations * depths];
292            for (int m = 0; m < maxAdjacent; m++) {
293                Direction dir = directions[m];
294                for (int x = 0; x < width; x++) {
295                    for (int y = 0; y < height; y++) {
296                        maps[0][m][y * width + x] = composite(x - dir.deltaX, y - dir.deltaY, 0, 0);
297                        maps[1][m][y * width + x] = composite(x + dir.deltaX, y + dir.deltaY, 0, 0);
298                    }
299                }
300            }
301            return maps;
302        }
303
304        @Override
305        public boolean isBlocked(int start, int direction, int[][][] neighbors, double[] map, double wall) {
306            if(direction < 4)
307                return !validate(start);
308            int[][] near = neighbors[0];
309            switch (direction)
310            {
311                case 4: //UP_LEFT
312                    return (near[0][start] < 0 || map[near[0][start]] >= wall)
313                            && (near[2][start] < 0 || map[near[2][start]] >= wall);
314                case 5: //UP_RIGHT
315                    return (near[0][start] < 0 || map[near[0][start]] >= wall)
316                            && (near[3][start] < 0 || map[near[3][start]] >= wall);
317                case 6: //DOWN_LEFT
318                    return (near[1][start] < 0 || map[near[1][start]] >= wall)
319                            && (near[2][start] < 0 || map[near[2][start]] >= wall);
320                default: //DOWN_RIGHT
321                    return (near[1][start] < 0 || map[near[1][start]] >= wall)
322                            && (near[3][start] < 0 || map[near[3][start]] >= wall);
323            }
324        }
325
326        @Override
327        public void portal(int[][][] neighbors, int inputPortal, int outputPortal, boolean twoWay) {
328            if(neighbors == null || !validate(inputPortal) || !validate(outputPortal)
329                    || neighbors.length != maxAdjacent)
330                return;
331            for (int d = 0; d < maxAdjacent; d++) {
332                for (int i = 0; i < width * height; i++) {
333                    if(neighbors[1][d][i] == inputPortal)
334                    {
335                        neighbors[1][d][i] = outputPortal;
336                    }
337                    else if(twoWay && neighbors[1][d][i] == outputPortal)
338                    {
339                        neighbors[1][d][i] = inputPortal;
340                    }
341
342                    if(neighbors[0][d][i] == outputPortal)
343                    {
344                        neighbors[0][d][i] = inputPortal;
345                    }
346                    else if(twoWay && neighbors[0][d][i] == inputPortal)
347                    {
348                        neighbors[0][d][i] = outputPortal;
349                    }
350                }
351            }
352        }
353
354        @Override
355        public IntDoubleOrderedMap addCostRule(char tile, double cost, boolean isRotation) {
356            costRules.put(tile, cost);
357            if(cost != costRules.defaultReturnValue())
358                standardCost = false;
359            return costRules;
360        }

362        @Override
363        public IntDoubleOrderedMap putAllVariants(IntDoubleOrderedMap map, int key, double value, int size) {
364            int baseX = key % width, baseY = key / width, comp;
365            if (key >= 0 && baseY < height) {
366                if (size < 0) {
367                    for (int x = size+1; x <= 0; x++) {
368                        for (int y = size+1; y <= 0; y++) {
369                            comp = composite(baseX + x, baseY + y, 0, 0);
370                            if(comp >= 0)
371                                map.put(comp, value);
372                        }
373                    }
374                } else {
375                    for (int x = 0; x < size; x++) {
376                        for (int y = 0; y < size; y++) {
377                            comp = composite(baseX + x, baseY + y, 0, 0);
378                            if(comp >= 0)
379                                map.put(comp, value);
380                        }
381                    }
382                }
383            }
384            return map;
385        }
386
387        @Override
388        public void putAllVariants(IntVLA list, double[] map, int key, double value, int size) {
389            int baseX = key % width, baseY = key / width, comp;
390            if (key >= 0 && baseY < height) {
391                if (size < 0) {
392                    if(list == null)
393                    {
394                        for (int x = size + 1; x <= 0; x++) {
395                            for (int y = size + 1; y <= 0; y++) {
396                                comp = composite(baseX + x, baseY + y, 0, 0);
397                                if (comp >= 0) {
398                                    map[comp] = value;
399                                }
400                            }
401                        }
402                    }
403                    else {
404                        for (int x = size + 1; x <= 0; x++) {
405                            for (int y = size + 1; y <= 0; y++) {
406                                comp = composite(baseX + x, baseY + y, 0, 0);
407                                if (comp >= 0 && !list.contains(comp)) {
408                                    list.add(comp);
409                                    map[comp] = value;
410                                }
411                            }
412                        }
413                    }
414                } else {
415                    if (list == null) {
416                        for (int x = 0; x < size; x++) {
417                            for (int y = 0; y < size; y++) {
418                                comp = composite(baseX + x, baseY + y, 0, 0);
419                                if (comp >= 0) {
420                                    map[comp] = value;
421                                }
422                            }
423                        }
424                    } else {
425                        for (int x = 0; x < size; x++) {
426                            for (int y = 0; y < size; y++) {
427                                comp = composite(baseX + x, baseY + y, 0, 0);
428                                if (comp >= 0 && !list.contains(comp)) {
429                                    list.add(comp);
430                                    map[comp] = value;
431                                }
432                            }
433                        }
434                    }
435                }
436            }
437        }

439        @Override
440        public void resetAllVariants(double[] map, int[] keys, int usable, double[] values, int size) {
441            int key;
442            for (int i = 0; i < usable && i < keys.length; i++) {
443                key = keys[i];
444                int baseX = key % width, baseY = key / width, comp;
445                if (key >= 0 && baseY < height) {
446                    if (size < 0) {
447                        for (int x = size + 1; x <= 0; x++) {
448                            for (int y = size + 1; y <= 0; y++) {
449                                comp = composite(baseX + x, baseY + y, 0, 0);
450                                if (comp >= 0) {
451                                    map[comp] = values[comp];
452                                }
453                            }
454                        }
455                    } else {
456                        for (int x = 0; x < size; x++) {
457                            for (int y = 0; y < size; y++) {
458                                comp = composite(baseX + x, baseY + y, 0, 0);
459                                if (comp >= 0) {
460                                    map[comp] = values[comp];
461                                }
462                            }
463                        }
464                    }
465                }
466            }
467        }

492    public static class RotationAdjacency extends Adjacency implements Serializable {
493        private static final long serialVersionUID = 0L;
494
495        private RotationAdjacency() {
496            this(20, 20, Measurement.MANHATTAN);
497        }
498        private int shift;
499        public RotationAdjacency(int width, int height, Measurement metric) {
500            this.width = width;
501            this.height = height;
502            measurement = metric;
503            if(metric == Measurement.MANHATTAN)
504            {
505                rotations = 4;
506                shift = 2;
507                directions = Direction.CARDINALS_CLOCKWISE;
508                invertAdjacent = new int[]{2, 3, 0, 1};
509            }
510            else
511            {
512                rotations = 8;
513                shift = 3;
514                directions = Direction.CLOCKWISE;
515                invertAdjacent = new int[]{4, 5, 6, 7, 0, 1, 2, 3};
516            }
517            depths = 1;
518            maxAdjacent = 3;
519            twoStepRule = false;
520            blockingRule = 2;
521            costRules.defaultReturnValue(1.0);
522            //invertAdjacent = new int[]{2, 1, 0};
523        }
524
525        @Override
526        public int extractX(int data) {
527            return (data >>> shift) % width;
528        }
529
530        @Override
531        public int extractY(int data) {
532            return (data >>> shift) / width;
533        }
534
535        @Override
536        public int extractR(int data) {
537            return data & (rotations - 1);
538        }
539
540        @Override
541        public int extractN(int data) {
542            return 0;
543        }
544
545        @Override
546        public int composite(int x, int y, int r, int n) {
547            if(x < 0 || y < 0 || x >= width || y >= height || r < 0 || r >= rotations)
548                return -1;
549            return ((y * width + x) << shift) | r;
550        }
551
552        @Override
553        public boolean validate(int data) {
554            return data >= 0 && extractY(data) < height;
555        }
556
557        @Override
558        public int[][][] neighborMaps() {
559            int[][][] maps = new int[2][maxAdjacent][width * height * rotations * depths];
560            int current;
561            Direction dir;
562            for (int r = 0; r < rotations; r++) {
563                dir = directions[r];
564                for (int x = 0; x < width; x++) {
565                    for (int y = 0; y < height; y++) {
566                        current = ((y * width + x) << shift) | r;
567                        maps[0][1][current] = composite(x - dir.deltaX, y - dir.deltaY, r, 0);
568                        maps[1][1][current] = composite(x + dir.deltaX, y + dir.deltaY, r, 0);
569                        maps[0][0][current] = maps[1][0][current] = composite(x, y, r - 1 & (rotations - 1), 0);
570                        maps[0][2][current] = maps[1][2][current] = composite(x, y, r + 1 & (rotations - 1), 0);
571                        //maps[0][composite(x, y, r - 1 & (rotations - 1), 0)] = current;
572                        //maps[2][composite(x, y, r + 1 & (rotations - 1), 0)] = current;
573                    }
574                }
575            }
576            return maps;
577        }
578
579        @Override
580        public boolean isBlocked(int start, int direction, int[][][] neighbors, double[] map, double wall) {
581            if(rotations <= 4 || (direction & 1) == 0)
582                return !validate(start);
583
584            return neighbors[0][0][start] < 0 || map[neighbors[0][0][start]] >= wall
585                    || neighbors[0][2][start] < 0 || map[neighbors[0][2][start]] >= wall;
586        }
587
588        @Override
589        public void portal(int[][][] neighbors, int inputPortal, int outputPortal, boolean twoWay) {
590            if(neighbors == null || !validate(inputPortal) || !validate(outputPortal)
591                    || neighbors.length != maxAdjacent)
592                return;
593            for (int i = 0; i < width * height * rotations; i++) {
594                if (neighbors[0][1][i] == inputPortal) {
595                    neighbors[0][1][i] = outputPortal;
596                } else if (twoWay && neighbors[0][1][i] == outputPortal) {
597                    neighbors[0][1][i] = inputPortal;
598                }
599
600                if (neighbors[1][1][i] == outputPortal) {
601                    neighbors[1][1][i] = inputPortal;
602                } else if (twoWay && neighbors[1][1][i] == inputPortal) {
603                    neighbors[1][1][i] = outputPortal;
