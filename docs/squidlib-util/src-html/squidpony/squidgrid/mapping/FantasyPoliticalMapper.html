012/**
013 * When you have a world map as produced by {@link WorldMapGenerator}, you may want to fill it with claims by various
014 * factions, where each faction may be hand-made and may consist of humans or some fantasy species, such as goblins,
015 * elves, or demons. This can assign contiguous areas of land to various factions, while acknowledging any preferences
016 * some species may have for specific types of land (elves may strongly prefer forest terrain, or flying demons may be
017 * the ideal residents for difficult mountainous terrain). This needs both a {@link WorldMapGenerator}, preferably
018 * {@link squidpony.squidgrid.mapping.WorldMapGenerator.TilingMap} or
019 * {@link squidpony.squidgrid.mapping.WorldMapGenerator.EllipticalMap} because they distribute area equally across the
020 * map, and a {@link squidpony.squidgrid.mapping.WorldMapGenerator.BiomeMapper} to allocate biomes and height/moisture
021 * info.
022 */
023@Beta
024public class FantasyPoliticalMapper implements Serializable {
025    private static final long serialVersionUID = 0L;

028     * Represents a group that claims territory on a world-map, such as a nation. Each Faction has a name, a short name
029     * that may be the same as the regular name, a FakeLanguageGen that would be used to generate place names in that
030     * Faction's territory, and a number of (possibly null) arrays and Sets that represent what terrains this Faction
031     * wants to claim and which it will avoid (e.g. elves may prefer claiming forests, while frost giants won't claim
032     * anywhere that's even remotely warm).
033     */

044        public UnorderedSet<String> preferredBiomes;

046         * An UnorderedSet of String keys, where each key is the name of a biome this Faction will never occupy.

281    /**
282     * For when you really don't care what arguments you give this, you can use this zero-parameter overload of
283     * generate() to produce a 128x128 {@link squidpony.squidgrid.mapping.WorldMapGenerator.TilingMap} world map with a
284     * {@link squidpony.squidgrid.mapping.WorldMapGenerator.SimpleBiomeMapper} biome mapper, filling it with 30 random
285     * Factions and trying to avoid unclaimed land. You may need to use {@link #atlas} to make sense of the randomly
286     * generated Factions. The seed will be random here.
287     * @return a 2D char array where each char can be used as a key into {@link #atlas} to find the Faction that claims it
288     */

292        biomeMapper = new WorldMapGenerator.SimpleBiomeMapper();
293        biomeMapper.makeBiomes(wmg);
294        return generate(rng.nextLong(), wmg, biomeMapper, null, 30, 1.0);

297    /**
298     * Generates a 2D char array that represents the claims to the land described by the WorldMapGenerator {@code wmg}
299     * and the BiomeMapper {@code biomeMapper} by various Factions, where {@link Faction} is an inner class.
300     * This starts with two default Factions for "Ocean" and "Wilderness" (unclaimed land) and adds randomly generated
301     * Factions to fill factionCount (the two default factions aren't counted against this limit). These Factions
302     * typically claim contiguous spans of land stretching out from a starting point that matches the Faction's
303     * preferences for biome, land height, heat, and moisture. If a Faction requires a biome (like "TropicalRainforest") 
304     * and the world has none of that type, then that Faction won't claim any land. If the WorldMapGenerator zooms in or
305     * out, you should call {@link #adjustZoom()} to get a different 2D char array that represents the zoomed-in area.

307     * unclaimed.

310     * @param biomeMapper a WorldMapGenerator.BiomeMapper, which must have been initialized with wmg and refer to the same world
311     * @param factionCount the number of factions to have claiming land; cannot be negative or more than 253
312     * @return a 2D char array where each char can be used as a key into {@link #atlas} to find the Faction that claims it

340    /**
341     * Generates a 2D char array that represents the claims to the land described by the WorldMapGenerator {@code wmg}
342     * and the BiomeMapper {@code biomeMapper} by various Factions, where {@link Faction} is an inner class.
343     * This starts with two default Factions for "Ocean" and "Wilderness" (unclaimed land) and adds all of
344     * {@code factions} until {@code factionCount} is reached; if it isn't reached, random Factions will be generated to
345     * fill factionCount (the two default factions aren't counted against this limit). These Factions typically claim
346     * contiguous spans of land stretching out from a starting point that matches the Faction's preferences for biome,
347     * land height, heat, and moisture. If a Faction requires a biome (like "TropicalRainforest") and the world has none
348     * of that type, then that Faction won't claim any land. If the WorldMapGenerator zooms in or out, you should call
349     * {@link #adjustZoom()} to get a different 2D char array that represents the zoomed-in area. This overload tries to
350     * claim the given {@code controlledFraction} of land in total, though 1.0 can rarely be reached unless there are
351     * many factions and few islands.
352     *
353     * @param seed the seed that determines how Factions will randomly spread around the world
354     * @param wmg a WorldMapGenerator, which must have produced a map by calling its generate() method
355     * @param biomeMapper a WorldMapGenerator.BiomeMapper, which must have been initialized with wmg and refer to the same world
356     * @param factions a Collection of {@link Faction} that will be copied, shuffled and used before adding any random Factions
357     * @param factionCount the number of factions to have claiming land; cannot be negative or more than 253
358     * @param controlledFraction between 0.0 and 1.0 inclusive; higher means more land has a letter, lower has more '%'
359     * @return a 2D char array where each char can be used as a key into {@link #atlas} to find the Faction that claims it
360     */
361    public char[][] generate(long seed, WorldMapGenerator wmg, WorldMapGenerator.BiomeMapper biomeMapper,
362                                  Collection<Faction> factions, int factionCount, double controlledFraction) {
363        rng.setState(seed);
364        factionCount = Math.abs(factionCount % 254);
365        Thesaurus th = new Thesaurus(rng.nextLong());
366        th.addKnownCategories();
367        ArrayList<Faction> fact = factions == null ? new ArrayList<Faction>() : rng.shuffle(factions);
368        for (int i = fact.size(); i < factionCount; i++) {
369            String name = th.makeNationName(), shortName = th.latestGenerated;
370            FakeLanguageGen lang;
371            if(th.randomLanguages == null || th.randomLanguages.isEmpty())
372                lang = FakeLanguageGen.randomLanguage(rng);
373            else
374                lang = th.randomLanguages.get(0);
375            fact.add(new Faction(lang, name, shortName));
376        }
377        if(factionCount > 0)
378            rng.shuffleInPlace(fact);
379        fact.add(0, new Faction(FakeLanguageGen.DEMONIC, "The Lost Wilderness", "Wilderness"));
380        fact.add(0, new Faction(FakeLanguageGen.ALIEN_O, "The Vast Domain of the Seafolk", "Seafolk", new String[]{"Ocean"}));
381        atlas = new OrderedMap<>(letters.subList(0, factionCount + 2), fact.subList(0, factionCount + 2), 0.5f);
382        this.wmg = wmg;
383        this.biomeMapper = biomeMapper;
384        width = wmg.width;
385        height = wmg.height;
386        GreasedRegion land = new GreasedRegion(wmg.heightCodeData, 4, 999);
387        politicalMap = land.toChars('%', '~');
388        int controlled = (int) (land.size() * Math.max(0.0, Math.min(1.0, controlledFraction)));
389
390        int[] centers = land.copy().randomScatter(rng, (int) (Math.sqrt(width * height) * 0.1 + 0.999), factionCount).asTightEncoded();
391        int cen, cx, cy, cx2, cy2, biome, high, hot, moist, count = centers.length, re;
392        String biomeName;
393        String[] biomeTable = biomeMapper.getBiomeNameTable();
394        int[] reorder = new int[count];
395        Faction current;
396        int[] factionIndices = new int[count];
397        for (int c = 0; c < count; c++) {
398            cen = centers[c];
399            cx = cen % width;
400            cy = cen / width;
401            biome = biomeMapper.getBiomeCode(cx, cy);
402            biomeName = biomeTable[biome];
403            high = wmg.heightCodeData[cx][cy];
404            hot = biomeMapper.getHeatCode(cx, cy);
405            moist = biomeMapper.getMoistureCode(cx, cy);
406            rng.randomOrdering(count, reorder);
407            PER_FACTION:
408            for (int i = 0; i < count; i++) {
409                current = fact.get(re = reorder[i]);
410                if(current.preferredBiomes == null || current.preferredBiomes.contains(biomeName))
411                {
412                    factionIndices[c] = re;
413                    break;
414                }
415                if(current.blockedBiomes != null && current.blockedBiomes.contains(biomeName))
416                    continue;
417                if(current.preferredHeight != null)
418                {
419                    for (int j = 0; j < current.preferredHeight.length; j++) {
420                        if(high == current.preferredHeight[j])
421                        {
422                            factionIndices[c] = re;
423                            break PER_FACTION;
424                        }
425                    }
426                }
427                if(current.preferredHeat != null)
428                {
429                    for (int j = 0; j < current.preferredHeat.length; j++) {
430                        if(hot == current.preferredHeat[j])
431                        {
432                            factionIndices[c] = re;
433                            break PER_FACTION;
434                        }
435                    }
436                }
437                if(current.preferredMoisture != null)
438                {
439                    for (int j = 0; j < current.preferredMoisture.length; j++) {
440                        if(moist == current.preferredMoisture[j])
441                        {
442                            factionIndices[c] = re;
443                            break PER_FACTION;
444                        }
445                    }
446                }
447            }
448        }
449        IntVLA[] fresh = new IntVLA[count];
450        int filled = 0;
451        boolean hasFresh = false;
452        int approximateArea = (controlled * 4) / (count * 3);
453        char[] keys = new char[count];
454        double[] biases = new double[count];
455        for (int i = 0; i < count; i++) {
456            fresh[i] = new IntVLA(approximateArea);
457            cen = centers[i];
458            fresh[i].add(cen);
459            cx = cen % width;
460            cy = cen / width;
461            politicalMap[cx][cy] = keys[i] = atlas.keyAt(factionIndices[i] + 2);
462            biases[i] = rng.nextDouble() * rng.nextDouble() + rng.nextDouble() + 0.03125;
463            hasFresh = true;
464        }
465        Direction[] dirs = Direction.CARDINALS;
466        IntVLA currentFresh;
467        GreasedRegion anySpillMap = new GreasedRegion(width, height),
468                anyFreshMap = new GreasedRegion(width, height);
469
470        while (hasFresh && filled < controlled) {
471            hasFresh = false;
472            for (int i = 0; i < count && filled < controlled; i++) {
473                currentFresh = fresh[i];
474                if (currentFresh.isEmpty())
475                    continue;
476                else
477                    hasFresh = true;
478                if (rng.nextDouble() < biases[i]) {
479                    int index = rng.nextIntHasty(currentFresh.size), cell = currentFresh.get(index);
480                    cx = cell % width;
481                    cy = cell / width;
482
483
484                    politicalMap[cx][cy] = keys[i];
485                    filled++;
486                    anySpillMap.insert(cx, cy);
487
488                    for (int d = 0; d < dirs.length; d++) {
489                        cx2 = wmg.wrapX(cx + dirs[d].deltaX, cy);
490                        cy2 = wmg.wrapY(cx, cy + dirs[d].deltaY);
491                        if (cx == cx2 && cy == cy2)
492                            continue;
493                        if (land.contains(cx2, cy2) && !anySpillMap.contains(cx2, cy2)) {
494                            if(!anyFreshMap.contains(cx2, cy2)) {
495                                currentFresh.add(cx2 + cy2 * width);
496                                anyFreshMap.insert(cx2, cy2);
497                            }
498                        }
499                    }
500                    currentFresh.removeIndex(index);
501                    anyFreshMap.remove(cx, cy);
502                }
503            }
504        }
505        zoomedMap = ArrayTools.copy(politicalMap);
506        return politicalMap;
507    }

509    /**
510     * If the WorldMapGenerator used by
511     * {@link #generate(long, WorldMapGenerator, WorldMapGenerator.BiomeMapper, Collection, int, double)} zooms in or
512     * out, you can call this method to make the {@link #zoomedMap} 2D char array match its zoom. The world-scale map,
513     * {@link #politicalMap}, will remain unchanged unless generate() is called again, but zoomedMap will change each
514     * time either generate() or adjustZoom() is called. This method isn't 100% precise on how it places borders; for
515     * aesthetic reasons, the borders are tattered with {@link GreasedRegion#fray(double)} so they don't look like a
516     * wall of angular bubbles. Using fray() at each level of zoom is quasi-random, so if you zoom in on the same
517     * sequence of points on two different occasions, the change from fray() will be the same, but it may be slightly
518     * different if any point of zoom is different.
519     * @return a direct reference to {@link #zoomedMap}, which will hold the correctly-zoomed version of {@link #politicalMap}
520     */
521    public char[][] adjustZoom() {
522        if(wmg.zoom <= 0)
523        {
524            return ArrayTools.insert(politicalMap, zoomedMap, 0, 0);
525        }
526        ArrayTools.fill(zoomedMap, ' ');
527        char c;
528        int stx = Math.min(Math.max((wmg.zoomStartX - (width  >> 1)) / ((2 << wmg.zoom) - 2), 0), width ),
529                sty = Math.min(Math.max((wmg.zoomStartY - (height >> 1)) / ((2 << wmg.zoom) - 2), 0), height);
530        GreasedRegion nation = new GreasedRegion(wmg.landData);
531        GreasedRegion fillable = new GreasedRegion(politicalMap, '~').not();
532        for (int i = 0; i < wmg.zoom; i++) {
533            fillable.zoom(stx, sty);
534        }
535        fillable.flood(nation, width + height);
536        for (int i = 1; i < atlas.size(); i++) {
537            nation.refill(politicalMap, c = atlas.keyAt(i));
538            if(nation.isEmpty()) continue;
539            for (int z = 0; z < wmg.zoom; z++) {
540                nation.zoom(stx, sty).expand8way().expand().fray(0.5);
541            }
542            fillable.andNot(nation);
543            nation.intoChars(zoomedMap, c);
544        }
545        for (int i = 1; i < atlas.size(); i++) {
546            nation.refill(zoomedMap, c = atlas.keyAt(i));
547            if(nation.isEmpty()) continue;
548            nation.flood(fillable, 4 << wmg.zoom).intoChars(zoomedMap, c);
549        }
550        nation.refill(wmg.heightCodeData, 4, 999).and(new GreasedRegion(zoomedMap, ' ')).intoChars(zoomedMap, '%');
551        nation.refill(wmg.heightCodeData, -999, 4).intoChars(zoomedMap, '~');
552        return zoomedMap;
553    }
