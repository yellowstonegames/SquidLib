014/**
015 * A replica of the previous API for DungeonGenerator. DungeonGenerator is the primary way to create a more-complete
016 * dungeon, layering different effects and modifications on top of a DungeonBoneGen's dungeon. This keeps the older API
017 * for DungeonGenerator around for games that relied on its behavior and seeds, but newer work should use the non-legacy
018 * DungeonGenerator class since it is significantly more precise in its results. The only difference between the two API
019 * versions is that LegacyDungeonGenerator uses 2/3 of the value it is given for addWater and addGrass, but is still not
020 * very accurate in how much of that value it uses, while DungeonGenerator gets very close to exactly the value it is
021 * given for those two methods. This class is deprecated in favor of DungeonGenerator.
022 * <br>
023 * The main technique for using this is simple: Construct a DungeonGenerator, usually with the desired width and height,
024 * then call any feature adding methods that you want in the dungeon, like addWater(), addTraps, addGrass(), or
025 * addDoors(). Some of these take different parameters, like addDoors() which need to know if it should check openings
026 * that are two cells wide to add a door and a wall to, or whether it should only add doors to single-cell openings.
027 * Then call generate() to get a char[][] with the desired dungeon map, using a fixed repertoire of chars to represent
028 * the different features. After calling generate(), you can safely get the values from the stairsUp and stairsDown
029 * fields, which are Coords that should be a long distance from each other but connected in the dungeon. You may want
030 * to change those to staircase characters, but there's no requirement to do anything with them. The DungeonUtility
031 * field of this class, utility, is a convenient way of accessing the non-static methods in that class, such as
032 * randomFloor(), without needing to create another DungeonUtility (this class creates one, so you don't have to).
033 *
034 * @see squidpony.squidgrid.mapping.DungeonUtility this class exposes a DungeonUtility member; DungeonUtility also has many useful static methods
035 * @see squidpony.squidgrid.mapping.DungeonGenerator for more-precise behavior
036 *
037 * @author Eben Howard - http://squidpony.com - howard@squidpony.com
038 * @author Tommy Ettinger - https://github.com/tommyettinger
039 * @deprecated Prefer DungeonGenerator with its almost-equivalent API
040 * Created by Tommy Ettinger on 3/5/2016.
041 */

133    /**
134     * Turns the majority of the given percentage of floor cells into water cells, represented by '~'. Water will be
135     * clustered into a random number of pools, with more appearing if needed to fill (two thirds of) the percentage.
136     * Each pool will have randomized volume that should fill or get very close to filling two thirds of the requested
137     * percentage, unless the pools encounter too much tight space. If this DungeonGenerator previously had addWater
138     * called, the latest call will take precedence. No islands will be placed with this variant, but the edge of the
139     * water will be shallow, represented by ','.
140     * @param percentage the percentage of floor cells to fill with water; this can vary quite a lot. It is not
141     *                   intended to allow filling very high (over 66%) percentages of map with water, though you can do
142     *                   this by giving a percentage of between 100 and 150.
143     * @return this DungeonGenerator; can be chained
144     */
145    @Override
146    public DungeonGenerator addWater(int percentage)
147    {
148        if(percentage < 0) percentage = 0;
149        if(percentage > 150) percentage = 150;
150        if(fx.containsKey(FillEffect.WATER)) fx.remove(FillEffect.WATER);
151        fx.put(FillEffect.WATER, percentage);
152        return this;
153    }

314    /**
315     * Generate a char[][] dungeon with extra features given a baseDungeon that has already been generated.
316     * Typically, you want to call generate with a TilesetType or no argument for the easiest generation; this method
317     * is meant for adding features like water and doors to existing simple maps.
318     * This uses '#' for walls, '.' for floors, '~' for deep water, ',' for shallow water, '^' for traps, '+' for doors
319     * that provide horizontal passage, and '/' for doors that provide vertical passage.
320     * Use the addDoors, addWater, addGrass, and addTraps methods of this class to request these in the generated map.
321     * Also sets the fields stairsUp and stairsDown to two randomly chosen, distant, connected, walkable cells.
322     * @param baseDungeon a pre-made dungeon consisting of '#' for walls and '.' for floors
323     * @return a char[][] dungeon
324     */
325    @Override
326    public char[][] generate(char[][] baseDungeon)
327    {
328        if(!seedFixed)
329        {
330            rebuildSeed = rng.getState();
331        }
332        seedFixed = false;
333        char[][] map = DungeonBoneGen.wallWrap(baseDungeon);
334        DijkstraMap dijkstra = new DijkstraMap(map);
335        int frustrated = 0;
336        do {
337            dijkstra.clearGoals();
338            stairsUp = utility.randomFloor(map);
339            dijkstra.setGoal(stairsUp);
340            dijkstra.scan(null);
341            frustrated++;
342        }while (dijkstra.getMappedCount() < width + height && frustrated < 15);
343        double maxDijkstra = 0.0;
344        for (int i = 0; i < width; i++) {
345            for (int j = 0; j < height; j++) {
346                if(dijkstra.gradientMap[i][j] >= DijkstraMap.FLOOR) {
347                    map[i][j] = '#';
348                }
349                else if(dijkstra.gradientMap[i][j] > maxDijkstra) {
350                    maxDijkstra = dijkstra.gradientMap[i][j];
351                }
352            }
353        }
354        stairsDown = CoordPacker.singleRandom(CoordPacker.pack(dijkstra.gradientMap, maxDijkstra * 0.7,
355                DijkstraMap.FLOOR), rng);
356        /*
357        OUTER_LOOP:
358        for (int i = 0; i < width; i++) {
359            for (int j = 0; j < height; j++) {
360                if (dijkstra.gradientMap[(i + wmod) % width][(j + hmod) % height] < DijkstraMap.FLOOR &&
361                        dijkstra.gradientMap[(i + wmod) % width][(j + hmod) % height] > maxDijkstra * 0.7) {
362                    stairsDown = Coord.get((i + wmod) % width, (j + hmod) % height);
363                    break OUTER_LOOP;
364                }
365            }
366        }*/
367        return innerGenerate(map);
368    }

370    /**
371     * Generate a char[][] dungeon with extra features given a baseDungeon that has already been generated, and that
372     * already has staircases represented by greater than and less than signs.
373     * Typically, you want to call generate with a TilesetType or no argument for the easiest generation; this method
374     * is meant for adding features like water and doors to existing simple maps.
375     * This uses '#' for walls, '.' for floors, '~' for deep water, ',' for shallow water, '^' for traps, '+' for doors
376     * that provide horizontal passage, and '/' for doors that provide vertical passage.
377     * Use the addDoors, addWater, addGrass, and addTraps methods of this class to request these in the generated map.
378     * Also sets the fields stairsUp and stairsDown to null, and expects stairs to be already handled.
379     * @param baseDungeon a pre-made dungeon consisting of '#' for walls and '.' for floors, with stairs already in
380     * @return a char[][] dungeon
381     */
382    @Override
383    public char[][] generateRespectingStairs(char[][] baseDungeon) {
384        if(!seedFixed)
385        {
386            rebuildSeed = rng.getState();
387        }
388        seedFixed = false;
389        char[][] map = DungeonBoneGen.wallWrap(baseDungeon);
390        DijkstraMap dijkstra = new DijkstraMap(map);
391        stairsUp = null;
392        stairsDown = null;
393
394        dijkstra.clearGoals();
395        Coord[] stairs = allPacked(unionPacked(pack(map, '<'), pack(map, '>')));
396        for (Coord s : stairs) {
397            dijkstra.setGoal(s);
398        }
399        dijkstra.scan(null);
400        for (int i = 0; i < width; i++) {
401            for (int j = 0; j < height; j++) {
402                if (dijkstra.gradientMap[i][j] >= DijkstraMap.FLOOR) {
403                    map[i][j] = '#';
404                }
405            }
406        }
407        return innerGenerate(map);
408    }

410    private char[][] innerGenerate(char[][] map)
411    {
412
413        OrderedSet<Coord> floors = new OrderedSet<>();
414        OrderedSet<Coord> doorways;
415        OrderedSet<Coord> hazards = new OrderedSet<>();
416        Coord temp;
417        boolean doubleDoors = false;
418        int doorFill = 0;
419        int waterFill = 0;
420        int grassFill = 0;
421        int trapFill = 0;
422        double boulderFill = 0.0;
423        int islandSpacing = 0;
424        if(fx.containsKey(FillEffect.DOORS))
425        {
426            doorFill = fx.get(FillEffect.DOORS);
427            if(doorFill < 0)
428            {
429                doubleDoors = true;
430                doorFill *= -1;
431            }
432        }
433        if(fx.containsKey(FillEffect.GRASS)) {
434            grassFill = fx.get(FillEffect.GRASS);
435        }
436        if(fx.containsKey(FillEffect.WATER)) {
437            waterFill = fx.get(FillEffect.WATER);
438        }
439        if(fx.containsKey(FillEffect.BOULDERS)) {
440            boulderFill = fx.get(FillEffect.BOULDERS) * 0.01;
441        }
442        if(fx.containsKey(FillEffect.ISLANDS)) {
443            islandSpacing = fx.get(FillEffect.ISLANDS);
444        }
445        if(fx.containsKey(FillEffect.TRAPS)) {
446            trapFill = fx.get(FillEffect.TRAPS);
447        }
448
449        double floorRate = 1.0, waterRate = waterFill / 150.0, grassRate = grassFill / 150.0;
450        if(waterRate + grassRate > 1.0)
451        {
452            waterRate /= (waterFill + grassFill) / 100.0;
453            grassRate /= (waterFill + grassFill) / 100.0;
454        }
455        floorRate -= waterRate + grassRate;
456        doorways = viableDoorways(doubleDoors, map);
457
458        OrderedSet<Coord> obstacles = new OrderedSet<>(doorways.size() * doorFill / 100);
459        if(doorFill > 0)
460        {
461            int total = doorways.size() * doorFill / 100;
462
463            BigLoop:
464            for(int i = 0; i < total; i++)
465            {
466                Coord entry = doorways.randomItem(rng);//toArray(new Coord[doorways.size()])[rng.nextInt(doorways.size())];
467                if(map[entry.x][entry.y] == '<' || map[entry.x][entry.y] == '>')
468                    continue;
469                if(map[entry.x - 1][entry.y] != '#' && map[entry.x + 1][entry.y] != '#')
470                {
471                    map[entry.x][entry.y] = '+';
472                }
473                else {
474                    map[entry.x][entry.y] = '/';
475                }
476                obstacles.add(entry);
477                Coord[] adj = new Coord[]{Coord.get(entry.x + 1, entry.y), Coord.get(entry.x - 1, entry.y),
478                        Coord.get(entry.x, entry.y + 1), Coord.get(entry.x, entry.y - 1)};
479                for(Coord near : adj) {
480                    if (doorways.contains(near)) {
481                        map[near.x][near.y] = '#';
482                        obstacles.add(near);
483                        doorways.remove(near);
484                        i++;
485                        doorways.remove(entry);
486                        continue BigLoop;
487                    }
488                }
489                doorways.remove(entry);
490            }
491        }
492        if (boulderFill > 0.0) {
493            short[] walls = unionPacked(unionPacked(pack(map, '>'), pack(map, '<')), pack(map, '#'));
494            short[] more = expand(walls, 1, width, height);
495            short[] rect = rectangle(width, height);
496            short[] viable = differencePacked(rect, more);
497            Coord[] boulders = randomSample(viable, boulderFill, rng);
498            for (int i = 0; i < boulders.length; i++) {
499                map[boulders[i].x][boulders[i].y] = '#';
500            }
501        }
502
503
504        if(trapFill > 0) {
505            for (int x = 1; x < map.length - 1; x++) {
506                for (int y = 1; y < map[x].length - 1; y++) {
507                    temp = Coord.get(x, y);
508                    if (map[x][y] == '.' && !obstacles.contains(temp)) {
509                        floors.add(Coord.get(x, y));
510                        int ctr = 0;
511                        if (map[x + 1][y] != '#') ++ctr;
512                        if (map[x - 1][y] != '#') ++ctr;
513                        if (map[x][y + 1] != '#') ++ctr;
514                        if (map[x][y - 1] != '#') ++ctr;
515                        if (map[x + 1][y + 1] != '#') ++ctr;
516                        if (map[x - 1][y + 1] != '#') ++ctr;
517                        if (map[x + 1][y - 1] != '#') ++ctr;
518                        if (map[x - 1][y - 1] != '#') ++ctr;
519                        if (ctr >= 5) hazards.add(Coord.get(x, y));
520                    }
521                }
522            }
523        }
524
525        MultiSpill ms = new MultiSpill(map, Spill.Measurement.MANHATTAN, rng);
526        OrderedMap<Coord, Double> fillers = new OrderedMap<>(128);
527        ArrayList<Coord> dots = PoissonDisk.sampleMap(map, Math.min(width, height) / 8f, rng, '#', '+', '/','<', '>');
528        for (int i = 0; i < dots.size(); i++) {
529            switch (i % 3) {
530                case 0:
531                    fillers.put(dots.get(i), floorRate);
532                    break;
533                case 1:
534                    fillers.put(dots.get(i), waterRate);
535                    break;
536                case 2:
537                    fillers.put(dots.get(i), grassRate);
538                    break;
539            }
540        }
541
542        ArrayList<ArrayList<Coord>> fills = ms.start(fillers, -1, null);
543
544        int ctr = 0;
545        for(ArrayList<Coord> pts : fills)
546        {
547            switch (ctr % 3) {
548                case 0:
549                    break;
550                case 1:
551                    floors.removeAll(pts);
552                    hazards.removeAll(pts);
553                    obstacles.addAll(pts);
554                    for (int x = 1; x < width - 1; x++) {
555                        for (int y = 1; y < height - 1; y++) {
556                            if (ms.spillMap[x][y] == ctr && map[x][y] != '<' && map[x][y] != '>') {
557                                map[x][y] = '~';
558                            }
559                        }
560                    }
561                    for (int x = 1; x < width - 1; x++) {
562                        for (int y = 1; y < height - 1; y++) {
563                            if (ms.spillMap[x][y] == ctr) {
564                                if(map[x][y] != '<' && map[x][y] != '>' &&
565                                        (map[x - 1][y] == '.' || map[x + 1][y] == '.' ||
566                                                map[x][y - 1] == '.' || map[x][y + 1] == '.'))
567                                    map[x][y] = ',';
568                            }
569                        }
570                    }
571                    if(islandSpacing > 1) {
572                        ArrayList<Coord> islands = PoissonDisk.sampleMap(map, 1f * islandSpacing, rng, '#', '.', '"', '+', '/', '^', '<', '>');
573                        for (Coord c : islands) {
574                            map[c.x][c.y] = '.';
575                            if (map[c.x - 1][c.y] != '#' && map[c.x - 1][c.y] != '<' && map[c.x - 1][c.y] != '>')
576                                map[c.x - 1][c.y] = ',';
577                            if (map[c.x + 1][c.y] != '#' && map[c.x + 1][c.y] != '<' && map[c.x + 1][c.y] != '>')
578                                map[c.x + 1][c.y] = ',';
579                            if (map[c.x][c.y-1] != '#' && map[c.x][c.y-1] != '<' && map[c.x][c.y-1] != '>')
580                                map[c.x][c.y - 1] = ',';
581                            if (map[c.x][c.y+1] != '#' && map[c.x][c.y+1] != '<' && map[c.x][c.y+1] != '>')
582                                map[c.x][c.y + 1] = ',';
583                        }
584                    }
585
586                    break;
587                case 2:
588                    floors.removeAll(pts);
589                    hazards.removeAll(pts);
590                    obstacles.addAll(pts);
591                    for (int x = 1; x < width - 1; x++) {
592                        for (int y = 1; y < height - 1; y++) {
593                            if (ms.spillMap[x][y] == ctr && map[x][y] == '.') {
594                                map[x][y] = '"';
595                            }
596                        }
597                    }
598                    break;
599            }
600            ctr++;
601        }
602
603        if(trapFill > 0)
604        {
605            int total = hazards.size() * trapFill / 100;
606
607            for(int i = 0; i < total; i++)
608            {
609                Coord entry = hazards.randomItem(rng);//.toArray(new Coord[hazards.size()])[rng.nextInt(hazards.size())];
610                if(map[entry.x][entry.y] == '<' || map[entry.x][entry.y] == '<')
611                    continue;
612                map[entry.x][entry.y] = '^';
613                hazards.remove(entry);
614            }
615        }
616
617        dungeon = map;
618        return map;
