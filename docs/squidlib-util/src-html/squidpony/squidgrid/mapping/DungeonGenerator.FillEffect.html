011/**
012 * The primary way to create a more-complete dungeon, layering different effects and modifications on top of
013 * a DungeonBoneGen's dungeon or another dungeon without such effects. Also ensures only connected regions of the map
014 * are used by filling unreachable areas with walls, and can find far-apart staircase positions if generate() is used or
015 * can keep existing staircases in a map if generateRespectingStairs() is used.
016 * <br>
017 * The main technique for using this is simple: Construct a DungeonGenerator, usually with the desired width and height,
018 * then call any feature adding methods that you want in the dungeon, like addWater(), addTraps, addGrass(), or
019 * addDoors(). Some of these take different parameters, like addDoors() which need to know if it should check openings
020 * that are two cells wide to add a door and a wall to, or whether it should only add doors to single-cell openings.
021 * Then call generate() to get a char[][] with the desired dungeon map, using a fixed repertoire of chars to represent
022 * the different features. After calling generate(), you can safely get the values from the stairsUp and stairsDown
023 * fields, which are Coords that should be a long distance from each other but connected in the dungeon. You may want
024 * to change those to staircase characters, but there's no requirement to do anything with them. It's recommended that
025 * you keep the resulting char[][] maps in some collection that can be saved, since DungeonGenerator only stores a
026 * temporary copy of the most recently-generated map. The DungeonUtility field of this class, utility, is a convenient
027 * way of accessing the non-static methods in that class, such as randomFloor(), without needing to create another
028 * DungeonUtility (this class creates one, so you don't have to).
029 * <br>
030 * Previews for the kinds of dungeon this generates, given a certain argument to generate():
031 * <ul>
032 *     <li>Using TilesetType.DEFAULT_DUNGEON (text, click "Raw", may need to zoom out): https://gist.github.com/tommyettinger/a3bd413b903f2e103541</li>
033 *     <li>Using TilesetType.DEFAULT_DUNGEON (graphical, scroll down and to the right): http://tommyettinger.github.io/home/PixVoxel/dungeon/dungeon.html</li>
034 *     <li>Using SerpentMapGenerator.generate()  (text, click "Raw", may need to zoom out): https://gist.github.com/tommyettinger/93b47048fc8a209a9712</li>
035 * </ul>
036 * <br>
037 * As of March 6, 2016, the algorithm this uses to place water and grass was swapped for a more precise version. You no
038 * longer need to give this 150% in addWater or addGrass to effectively produce 100% water or grass, and this should be
039 * no more than 1.1% different from the percentage you request for any effects. If you need to reproduce dungeons with
040 * the same seed and get the same (imprecise) results as before this change, it's probably not possible unless you save
041 * the previously-generated char[][] dungeons, since several other things may have changed as well.
042 * @see squidpony.squidgrid.mapping.DungeonUtility this class exposes a DungeonUtility member; DungeonUtility also has many useful static methods
043 *
044 * @author Eben Howard - http://squidpony.com - howard@squidpony.com
045 * @author Tommy Ettinger - https://github.com/tommyettinger
046 */
047public class DungeonGenerator implements IDungeonGenerator {

163     * creation of the dungeon (if width and height are 300 and 300, the Coord pool will be 300x300; if width and height

165     * @param width The width of the dungeon in cells

173    /**
174     * Make a DungeonGenerator with the given height, width, and RNG. Use this if you want to seed the RNG. If width or
175     * height is greater than 256, then this will expand the Coord pool from its 256x256 default so it stores a
176     * reference to each Coord that might be used in the creation of the dungeon (if width and height are 300 and 300,
177     * the Coord pool will be 300x300; if width and height are 500 and 100, the Coord pool will be 500x256 because it
178     * won't shrink below the default size of 256x256).
179     * @param width The width of the dungeon in cells
180     * @param height The height of the dungeon in cells
181     * @param rng The RNG to use for all purposes in this class; if it is a StatefulRNG, then it will be used as-is,
182     *            but if it is not a StatefulRNG, a new StatefulRNG will be used, randomly seeded by this parameter
183     */
184    public DungeonGenerator(int width, int height, IRNG rng)
185    {
186        Coord.expandPoolTo(width, height);
187        this.rng = (rng instanceof StatefulRNG) ? (StatefulRNG) rng : new StatefulRNG(rng.nextLong());
188        gen = new DungeonBoneGen(this.rng);
189        utility = new DungeonUtility(this.rng);
190        rebuildSeed = this.rng.getState();
191        this.height = height;
192        this.width = width;
193        fx = new EnumMap<>(FillEffect.class);
194    }

213    /**
214     * Turns the majority of the given percentage of floor cells into water cells, represented by '~'. Water will be
215     * clustered into a random number of pools, with more appearing if needed to fill the percentage.
216     * Each pool will have randomized volume that should fill or get very close to filling the requested
217     * percentage, unless the pools encounter too much tight space. If this DungeonGenerator previously had addWater
218     * called, the latest call will take precedence. No islands will be placed with this variant, but the edge of the
219     * water will be shallow, represented by ','.
220     * @param percentage the percentage of floor cells to fill with water
221     * @return this DungeonGenerator; can be chained
222     */
223    public DungeonGenerator addWater(int percentage)
224    {
225        if(percentage < 0) percentage = 0;
226        if(percentage > 100) percentage = 100;
227        if(fx.containsKey(FillEffect.WATER)) fx.remove(FillEffect.WATER);
228        fx.put(FillEffect.WATER, percentage);
229        return this;
230    }
231    /**
232     * Turns the majority of the given percentage of floor cells into water cells, represented by '~'. Water will be
233     * clustered into a random number of pools, with more appearing if needed to fill the percentage. Each pool will
234     * have randomized volume that should fill or get very close to filling the requested percentage,
235     * unless the pools encounter too much tight space. If this DungeonGenerator previously had addWater called, the
236     * latest call will take precedence. If islandSpacing is greater than 1, then this will place islands of floor, '.',
237     * surrounded by shallow water, ',', at about the specified distance with Euclidean measurement.
238     * @param percentage the percentage of floor cells to fill with water
239     * @param islandSpacing if greater than 1, islands will be placed randomly this many cells apart.
240     * @return this DungeonGenerator; can be chained
241     */
242    public DungeonGenerator addWater(int percentage, int islandSpacing)
243    {
244        if(percentage < 0) percentage = 0;
245        if(percentage > 100) percentage = 100;
246        if(fx.containsKey(FillEffect.WATER)) fx.remove(FillEffect.WATER);
247        fx.put(FillEffect.WATER, percentage);
248        if(fx.containsKey(FillEffect.ISLANDS)) fx.remove(FillEffect.ISLANDS);
249        if(islandSpacing > 1)
250            fx.put(FillEffect.ISLANDS, islandSpacing);
251        return this;
252    }

285    /**
286     * Turns the given percentage of viable doorways into doors, represented by '+' for doors that allow travel along
287     * the x-axis and '/' for doors that allow travel along the y-axis. If doubleDoors is true,
288     * 2-cell-wide openings will be considered viable doorways and will fill one cell with a wall, the other a door.
289     * If this DungeonGenerator previously had addDoors called, the latest call will take precedence.
290     * @param percentage the percentage of valid openings to corridors to fill with doors; should be between 10 and
291     *                   20 if you want doors to appear more than a few times, but not fill every possible opening.
292     * @param doubleDoors true if you want two-cell-wide openings to receive a door and a wall; false if only
293     *                    one-cell-wide openings should receive doors. Usually, this should be true.
294     * @return this DungeonGenerator; can be chained
295     */
296    public DungeonGenerator addDoors(int percentage, boolean doubleDoors)
297    {
298        if(percentage < 0) percentage = 0;
299        if(percentage > 100) percentage = 100;
300        if(doubleDoors) percentage *= -1;
301        if(fx.containsKey(FillEffect.DOORS)) fx.remove(FillEffect.DOORS);
302        fx.put(FillEffect.DOORS, percentage);
303        return this;
304    }

333    protected OrderedSet<Coord> removeAdjacent(OrderedSet<Coord> coll, Coord pt)
334    {
335        for(Coord temp : new Coord[]{Coord.get(pt.x + 1, pt.y), Coord.get(pt.x - 1, pt.y),
336                Coord.get(pt.x, pt.y + 1), Coord.get(pt.x, pt.y - 1)})
337        {
338            if(coll.contains(temp) && !(temp.x == pt.x && temp.y == pt.y))
339                coll.remove(temp);
340        }
341
342        return coll;
343    }
344    protected OrderedSet<Coord> removeAdjacent(OrderedSet<Coord> coll, Coord pt1, Coord pt2)
345    {
346
347        for(Coord temp : new Coord[]{Coord.get(pt1.x + 1, pt1.y), Coord.get(pt1.x - 1, pt1.y),
348                Coord.get(pt1.x, pt1.y + 1), Coord.get(pt1.x, pt1.y - 1),
349                Coord.get(pt2.x + 1, pt2.y), Coord.get(pt2.x - 1, pt2.y),
350                Coord.get(pt2.x, pt2.y + 1), Coord.get(pt2.x, pt2.y - 1),})
351        {
352            if(coll.contains(temp) && !(temp.x == pt1.x && temp.y == pt1.y) && !(temp.x == pt2.x && temp.y == pt2.y))
353                coll.remove(temp);
354        }
355
356        return coll;
357    }
358    protected OrderedSet<Coord> viableDoorways(boolean doubleDoors, char[][] map)
359    {
360        OrderedSet<Coord> doors = new OrderedSet<>();
361        OrderedSet<Coord> blocked = new OrderedSet<>(4);
362        DijkstraMap dm = new DijkstraMap(map, DijkstraMap.Measurement.EUCLIDEAN);
363        for(int x = 1; x < map.length - 1; x++) {
364            for (int y = 1; y < map[x].length - 1; y++) {
365                if(map[x][y] == '#')
366                    continue;
367                if (doubleDoors) {
368                    if (x >= map.length - 2 || y >= map[x].length - 2)
369                        continue;
370                    else {
371                        if (map[x+1][y] != '#' &&
372                                map[x + 2][y] == '#' && map[x - 1][y] == '#'
373                                && map[x][y + 1] != '#' && map[x][y - 1] != '#'
374                                && map[x+1][y + 1] != '#' && map[x+1][y - 1] != '#') {
375                            if (map[x + 2][y + 1] != '#' || map[x - 1][y + 1] != '#' || map[x + 2][y - 1] != '#' || map[x - 1][y - 1] != '#') {
376                                dm.resetMap();
377                                dm.clearGoals();
378                                dm.setGoal(x, y+1);
379                                blocked.clear();
380                                blocked.add(Coord.get(x, y));
381                                blocked.add(Coord.get(x + 1, y));
382                                dm.partialScan(null, 16, blocked);
383                                if(dm.gradientMap[x][y-1] < DijkstraMap.FLOOR)
384                                    continue;
385                                doors.add(Coord.get(x, y));
386                                doors.add(Coord.get(x + 1, y));
387                                doors = removeAdjacent(doors, Coord.get(x, y), Coord.get(x + 1, y));
388                                continue;
389                            }
390                        } else if (map[x][y+1] != '#' &&
391                                map[x][y + 2] == '#' && map[x][y - 1] == '#'
392                                && map[x + 1][y] != '#' && map[x - 1][y] != '#'
393                                && map[x + 1][y+1] != '#' && map[x - 1][y+1] != '#') {
394                            if (map[x + 1][y + 2] != '#' || map[x + 1][y - 1] != '#' || map[x - 1][y + 2] != '#' || map[x - 1][y - 1] != '#') {
395                                dm.resetMap();
396                                dm.clearGoals();
397                                dm.setGoal(x+1, y);
398                                blocked.clear();
399                                blocked.add(Coord.get(x, y));
400                                blocked.add(Coord.get(x, y+1));
401                                dm.partialScan(null, 16, blocked);
402                                if(dm.gradientMap[x-1][y] < DijkstraMap.FLOOR)
403                                    continue;
404                                doors.add(Coord.get(x, y));
405                                doors.add(Coord.get(x, y+1));
406                                doors = removeAdjacent(doors, Coord.get(x, y), Coord.get(x, y + 1));
407                                continue;
408                            }
409                        }
410                    }
411                }
412                if (map[x + 1][y] == '#' && map[x - 1][y] == '#' && map[x][y + 1] != '#' && map[x][y - 1] != '#') {
413                    if (map[x + 1][y + 1] != '#' || map[x - 1][y + 1] != '#' || map[x + 1][y - 1] != '#' || map[x - 1][y - 1] != '#') {
414                        dm.resetMap();
415                        dm.clearGoals();
416                        dm.setGoal(x, y+1);
417                        blocked.clear();
418                        blocked.add(Coord.get(x, y));
419                        dm.partialScan(null, 16, blocked);
420                        if(dm.gradientMap[x][y-1] < DijkstraMap.FLOOR)
421                            continue;
422                        doors.add(Coord.get(x, y));
423                        doors = removeAdjacent(doors, Coord.get(x, y));
424                    }
425                } else if (map[x][y + 1] == '#' && map[x][y - 1] == '#' && map[x + 1][y] != '#' && map[x - 1][y] != '#') {
426                    if (map[x + 1][y + 1] != '#' || map[x + 1][y - 1] != '#' || map[x - 1][y + 1] != '#' || map[x - 1][y - 1] != '#') {
427                        dm.resetMap();
428                        dm.clearGoals();
429                        dm.setGoal(x+1, y);
430                        blocked.clear();
431                        blocked.add(Coord.get(x, y));
432                        dm.partialScan(null, 16, blocked);
433                        if(dm.gradientMap[x-1][y] < DijkstraMap.FLOOR)
434                            continue;
435                        doors.add(Coord.get(x, y));
436                        doors = removeAdjacent(doors, Coord.get(x, y));
437                    }
438                }
439
440            }
441        }
442
443        return doors;
444    }

477    /**
478     * Generate a char[][] dungeon with extra features given a baseDungeon that has already been generated.
479     * Typically, you want to call generate with a TilesetType or no argument for the easiest generation; this method
480     * is meant for adding features like water and doors to existing simple maps.
481     * This uses '#' for walls, '.' for floors, '~' for deep water, ',' for shallow water, '^' for traps, '+' for doors
482     * that provide horizontal passage, and '/' for doors that provide vertical passage.
483     * Use the addDoors, addWater, addGrass, and addTraps methods of this class to request these in the generated map.
484     * Also sets the fields stairsUp and stairsDown to two randomly chosen, distant, connected, walkable cells.
485     * <br>
486     * Special behavior here: If tab characters are present in the 2D char array, they will be replaced with '.' in the
487     * final dungeon, but will also be tried first as valid staircase locations (with a high distance possible to travel
488     * away from the starting staircase). If no tab characters are present this will search for '.' floors to place
489     * stairs on, as normal. This tab-first behavior is useful in conjunction with some methods that establish a good
490     * path in an existing dungeon; an example is {@code DungeonUtility.ensurePath(dungeon, rng, '\t', '#');} then
491     * passing dungeon (which that code modifies) in as baseDungeon to this method.
492     * @param baseDungeon a pre-made dungeon consisting of '#' for walls and '.' for floors; may be modified in-place
493     * @return a char[][] dungeon
494     */
495    public char[][] generate(char[][] baseDungeon)
496    {
497        if(!seedFixed)
498        {
499            rebuildSeed = rng.getState();
500        }
501        seedFixed = false;
502        char[][] map = DungeonUtility.wallWrap(baseDungeon);
503        width = map.length;
504        height = map[0].length;
505        DijkstraMap dijkstra = new DijkstraMap(map);
506        int frustrated = 0;
507        do {
508            dijkstra.clearGoals();
509            stairsUp = utility.randomMatchingTile(map, '\t');
510            if(stairsUp == null) {
511                stairsUp = utility.randomFloor(map);
512                if (stairsUp == null) {
513                    frustrated++;
514                    continue;
515                }
516            }
517            dijkstra.setGoal(stairsUp);
518            dijkstra.scan(null, null);
519            frustrated++;
520        }while (dijkstra.getMappedCount() < width + height && frustrated < 8);
521        if(frustrated >= 8)
522        {
523            return generate();
524        }
525        double maxDijkstra = 0.0;
526        for (int i = 0; i < width; i++) {
527            for (int j = 0; j < height; j++) {
528                if (dijkstra.gradientMap[i][j] >= DijkstraMap.FLOOR) {
529                    map[i][j] = '#';
530                } else if (dijkstra.gradientMap[i][j] > maxDijkstra) {
531                    maxDijkstra = dijkstra.gradientMap[i][j];
532                }
533                if (map[i][j] == '\t') {
534                    map[i][j] = '.';
535                }
536            }
537        }
538        stairsDown = new GreasedRegion(dijkstra.gradientMap, maxDijkstra * 0.7,
539                DijkstraMap.FLOOR).singleRandom(rng);
540
541        return innerGenerate(map);
542    }

544    /**
545     * Generate a char[][] dungeon with extra features given a baseDungeon that has already been generated, and that
546     * already has staircases represented by greater than and less than signs.
547     * Typically, you want to call generate with a TilesetType or no argument for the easiest generation; this method
548     * is meant for adding features like water and doors to existing simple maps.
549     * This uses '#' for walls, '.' for floors, '~' for deep water, ',' for shallow water, '^' for traps, '+' for doors
550     * that provide horizontal passage, and '/' for doors that provide vertical passage.
551     * Use the addDoors, addWater, addGrass, and addTraps methods of this class to request these in the generated map.
552     * Also sets the fields stairsUp and stairsDown to null, and expects stairs to be already handled.
553     * @param baseDungeon a pre-made dungeon consisting of '#' for walls and '.' for floors, with stairs already in;
554     *                    may be modified in-place
555     * @return a char[][] dungeon
556     */
557    public char[][] generateRespectingStairs(char[][] baseDungeon) {
558        if(!seedFixed)
559        {
560            rebuildSeed = rng.getState();
561        }
562        seedFixed = false;
563        char[][] map = DungeonUtility.wallWrap(baseDungeon);
564        DijkstraMap dijkstra = new DijkstraMap(map);
565        stairsUp = null;
566        stairsDown = null;
567
568        dijkstra.clearGoals();
569        ArrayList<Coord> stairs = DungeonUtility.allMatching(map, '<', '>');
570        for (int j = 0; j < stairs.size(); j++) {
571            dijkstra.setGoal(stairs.get(j));
572        }
573        dijkstra.scan(null, null);
574        for (int i = 0; i < width; i++) {
575            for (int j = 0; j < height; j++) {
576                if (dijkstra.gradientMap[i][j] >= DijkstraMap.FLOOR) {
577                    map[i][j] = '#';
578                }
579            }
580        }
581        return innerGenerate(map);
582    }

587    private char[][] innerGenerate(char[][] map)
588    {
589        OrderedSet<Coord> doorways;
590        OrderedSet<Coord> hazards = new OrderedSet<>();
591        Coord temp;
592        boolean doubleDoors = false;
593        int floorCount = DungeonUtility.countCells(map, '.'),
594                doorFill = 0,
595                waterFill = 0,
596                grassFill = 0,
597                trapFill = 0,
598                boulderFill = 0,
599                islandSpacing = 0;
600        if(fx.containsKey(FillEffect.DOORS))
601        {
602            doorFill = fx.get(FillEffect.DOORS);
603            if(doorFill < 0)
604            {
605                doubleDoors = true;
606                doorFill *= -1;
607            }
608        }
609        if(fx.containsKey(FillEffect.GRASS)) {
610            grassFill = fx.get(FillEffect.GRASS);
611        }
612        if(fx.containsKey(FillEffect.WATER)) {
613            waterFill = fx.get(FillEffect.WATER);
614        }
615        if(fx.containsKey(FillEffect.BOULDERS)) {
616            boulderFill = fx.get(FillEffect.BOULDERS) * floorCount / 100;
617        }
618        if(fx.containsKey(FillEffect.ISLANDS)) {
619            islandSpacing = fx.get(FillEffect.ISLANDS);
620        }
621        if(fx.containsKey(FillEffect.TRAPS)) {
622            trapFill = fx.get(FillEffect.TRAPS);
623        }
624
625        doorways = viableDoorways(doubleDoors, map);
626
627        OrderedSet<Coord> obstacles = new OrderedSet<>(doorways.size() * doorFill / 100);
628        if(doorFill > 0)
629        {
630            int total = doorways.size() * doorFill / 100;
631
632            BigLoop:
633            for(int i = 0; i < total; i++)
634            {
635                Coord entry = rng.getRandomElement(doorways);
636                if(map[entry.x][entry.y] == '<' || map[entry.x][entry.y] == '>')
637                    continue;
638                if(map[entry.x - 1][entry.y] != '#' && map[entry.x + 1][entry.y] != '#')
639                {
640                    map[entry.x][entry.y] = '+';
641                }
642                else {
643                    map[entry.x][entry.y] = '/';
644                }
645                obstacles.add(entry);
646                Coord[] adj = new Coord[]{Coord.get(entry.x + 1, entry.y), Coord.get(entry.x - 1, entry.y),
647                        Coord.get(entry.x, entry.y + 1), Coord.get(entry.x, entry.y - 1)};
648                for(Coord near : adj) {
649                    if (doorways.contains(near)) {
650                        map[near.x][near.y] = '#';
651                        obstacles.add(near);
652                        doorways.remove(near);
653                        i++;
654                        doorways.remove(entry);
655                        continue BigLoop;
656                    }
657                }
658                doorways.remove(entry);
659            }
660        }
661        if (boulderFill > 0.0) {
662            /*
663            short[] floor = pack(map, '.');
664            short[] viable = retract(floor, 1, width, height, true);
665            ArrayList<Coord> boulders = randomPortion(viable, boulderFill, rng);
666            for (Coord boulder : boulders) {
667                map[boulder.x][boulder.y] = '#';
668            }
669            */
670            Coord[] boulders = new GreasedRegion(map, '.').retract8way(1).randomPortion(rng, boulderFill);
671            Coord t;
672            for (int i = 0; i < boulders.length; i++) {
673                t = boulders[i];
674                map[t.x][t.y] = '#';
675            }
676        }
677
678
679        if(trapFill > 0) {
680            for (int x = 1; x < map.length - 1; x++) {
681                for (int y = 1; y < map[x].length - 1; y++) {
682                    temp = Coord.get(x, y);
683                    if (map[x][y] == '.' && !obstacles.contains(temp)) {
684                        int ctr = 0;
685                        if (map[x + 1][y] != '#') ++ctr;
686                        if (map[x - 1][y] != '#') ++ctr;
687                        if (map[x][y + 1] != '#') ++ctr;
688                        if (map[x][y - 1] != '#') ++ctr;
689                        if (map[x + 1][y + 1] != '#') ++ctr;
690                        if (map[x - 1][y + 1] != '#') ++ctr;
691                        if (map[x + 1][y - 1] != '#') ++ctr;
692                        if (map[x - 1][y - 1] != '#') ++ctr;
693                        if (ctr >= 5) hazards.add(Coord.get(x, y));
694                    }
695                }
696            }
697        }
698        GreasedRegion floors = new GreasedRegion(map, '.'), working = new GreasedRegion(width, height);
699        floorCount = floors.size();
700        float waterRate = waterFill / 100.0f, grassRate = grassFill / 100.0f;
701        if(waterRate + grassRate > 1.0f)
702        {
703            waterRate /= (waterFill + grassFill) / 100.0f;
704            grassRate /= (waterFill + grassFill) / 100.0f;
705        }
706        int targetWater = Math.round(floorCount * waterRate),
707                targetGrass = Math.round(floorCount * grassRate),
708                sizeWaterPools = targetWater / rng.between(3, 6),
709                sizeGrassPools = targetGrass / rng.between(2, 5);
710
711        Coord[] scatter;
712        int remainingWater = targetWater, remainingGrass = targetGrass;
713        if(targetWater > 0) {
714            scatter = floors.quasiRandomSeparated(1.0 / 7.0);
715            rng.shuffleInPlace(scatter);
716            GreasedRegion allWater = new GreasedRegion(width, height);
717            for (int i = 0; i < scatter.length; i++) {
718                if (remainingWater > 5)
719                {
720                    if(!floors.contains(scatter[i]))
721                        continue;
722                    working.empty().insert(scatter[i]).spill(floors, rng.between(4, Math.min(remainingWater, sizeWaterPools)), rng);
723
724                    floors.andNot(working);
725                    remainingWater -= working.size();
726                    allWater.addAll(working);
727                } else
728                    break;
729            }
730
731            for (Coord pt : allWater) {
732                hazards.remove(pt);
733                //obstacles.add(pt);
734                if (map[pt.x][pt.y] != '<' && map[pt.x][pt.y] != '>')
735                    map[pt.x][pt.y] = '~';
736            }
737            for (Coord pt : allWater) {
738                if (map[pt.x][pt.y] != '<' && map[pt.x][pt.y] != '>' &&
739                        (map[pt.x - 1][pt.y] == '.' || map[pt.x + 1][pt.y] == '.' ||
740                                map[pt.x][pt.y - 1] == '.' || map[pt.x][pt.y + 1] == '.'))
741                    map[pt.x][pt.y] = ',';
742            }
743        }
744        if(targetGrass > 0) {
745            scatter = floors.quasiRandomSeparated(1.03/6.7);
746            rng.shuffleInPlace(scatter);
747            for (int i = 0; i < scatter.length; i++) {
748                if (remainingGrass > 5) //remainingGrass >= targetGrass * 0.02 &&
749                {
750                    working.empty().insert(scatter[i]).spill(floors, rng.between(4, Math.min(remainingGrass, sizeGrassPools)), rng);
751                    if (working.isEmpty())
752                        continue;
753                    floors.andNot(working);
754                    remainingGrass -= working.size();
755                    map = working.inverseMask(map, '"');
756                } else
757                    break;
758            }
759        }
760
761        if(islandSpacing > 1 && targetWater > 0) {
762            OrderedSet<Coord> islands = PoissonDisk.sampleMap(map, 1f * islandSpacing, rng, '#', '.', '"', '+', '/', '^', '<', '>');
763            for (Coord c : islands) {
764                map[c.x][c.y] = '.';
765                if (map[c.x - 1][c.y] != '#' && map[c.x - 1][c.y] != '<' && map[c.x - 1][c.y] != '>')
766                    map[c.x - 1][c.y] = ',';
767                if (map[c.x + 1][c.y] != '#' && map[c.x + 1][c.y] != '<' && map[c.x + 1][c.y] != '>')
768                    map[c.x + 1][c.y] = ',';
769                if (map[c.x][c.y - 1] != '#' && map[c.x][c.y - 1] != '<' && map[c.x][c.y - 1] != '>')
770                    map[c.x][c.y - 1] = ',';
771                if (map[c.x][c.y + 1] != '#' && map[c.x][c.y + 1] != '<' && map[c.x][c.y + 1] != '>')
772                    map[c.x][c.y + 1] = ',';
773            }
774        }
775
776        if(trapFill > 0)
777        {
778            int total = hazards.size() * trapFill / 100;
779
780            for(int i = 0; i < total; i++)
781            {
782                Coord entry = rng.getRandomElement(hazards);
783                if(map[entry.x][entry.y] == '<' || map[entry.x][entry.y] == '<')
784                    continue;
785                map[entry.x][entry.y] = '^';
786                hazards.remove(entry);
787            }
788        }
789
790        dungeon = map;
791        return map;
792
793    }

798     * will be used to generate the first dungeon if none has been made yet). You can pass the long this returns to
799     * the setState() method on this class' rng field, which assuming all other calls to generate a dungeon are
800     * identical, will ensure generate() or generateRespectingStairs() will produce the same dungeon output as the

806     * getState() and when this class generates a dungeon. Using getRebuildSeed() eliminates that confusion.
807     * @return a seed as a long that can be passed to setState() on this class' rng field to recreate a dungeon

814     * Provides a string representation of the latest generated dungeon.
815     *
816     * @return a printable string version of the latest generated dungeon.
817     */
818    @Override
819        public String toString() {
820        char[][] trans = new char[height][width];
821        for (int x = 0; x < width; x++) {
822            for (int y = 0; y < height; y++) {
823                trans[y][x] = dungeon[x][y];
824            }
825        }
826        StringBuffer sb = new StringBuffer();
827        for (int row = 0; row < height; row++) {
828            sb.append(trans[row]);
829            sb.append('\n');
830        }
831        return sb.toString();
832    }
